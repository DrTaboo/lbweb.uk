# Website Source Code

Hai!
This is the source code of my website hosted at [lbweb.uk](https://www.lbweb.uk)
I compiled this using GoHugo with the [Blowfish](https://github.com/nunocoracao/blowfish) theme. Everything outside of the theme folder is made by me or edited by me from the original config. 
Once edited files are pushed to github, a github worker action takes place deploying it onto one of my servers for hosting on my domain.

If you have any issues, please open a ticket [here](https://github.com/DrTaboo/lbweb.uk/issues).
Furthermore, you can contact me via [contactme@lbweb.uk](mailto:contactme@lbweb.uk).
